Deskripsi

Pak Dengklek mengajak Anda untuk bermain. Awalnya terdapat $K$ buah tumpukan batu yang dinomori dari $1$ hingga $K$. Tumpukan ke-$i$ memiliki batu sebanyak $A_i$.

Setiap pemain bermain secara bergiliran. Anda adalah pemain pada giliran pertama. Pada gilirannya, pemain mengambil setidaknya satu batu dari salah satu tumpukan dengan syarat jumlah batu yang diambil tidak lebih dari $N$ dan pada tumpukan tersebut masih terdapat batu yang cukup. Apabila seorang pemain tidak dapat melakukan langkah valid apa pun pada gilirannya, maka pemain tersebut dinyatakan kalah dan permainan berakhir.

Karena Anda adalah orang yang kompetitif, Anda tidak ingin kalah melawan Pak Dengklek. Oleh karena itu, Anda ingin membuat program yang dapat mengalahkan Pak Dengklek.

Informasi Tipe Soal

Soal ini bertipe "interaktif". Pada soal ini Anda akan berinteraksi dengan program penguji melalui standar masukan (stdin) dan standar keluaran (stdout). Perhatikan format interaksi di bawah ini dengan saksama.

Format Interaksi

Pada awalnya, nilai $K$, $N$, dan semua nilai $A_i$ akan diberikan dari standar masukan dalam format berikut: 

K N
A1 A2 ... AK

Kemudian, untuk setiap giliran, seorang pemain dapat mengambil $x$ buah batu dari tumpukan ke-$i$ dalam format berikut: 

i x

Apabila Pak Dengklek tidak dapat melakukan langkah yang valid, maka akan diberikan masukan dalam format berikut: 

-1 -1

Interaksi berakhir dengan masukan tersebut dan program Anda akan dianggap benar untuk kasus uji tersebut.

Contoh Interaksi 1

Pada contoh berikut, $K = 1, N = 2$, dan $A_1 = 4$.

Standar masukan Standar keluaran 
1 2
4

1 1

1 2

1 1

-1 -1

Contoh Interaksi 2

Pada contoh berikut, $K = 2, N = 2, A_1 = 2$, and $A_2 = 1$.

2 2
2 1

1 1

1 1

2 1

-1 -1

Subsoal

Untuk semua subsoal, berlaku:

  • $1 \leq K \leq 2$
  • $1 \leq A_i \leq 10^4$, untuk $1 \leq i \leq K$
  • $1 \leq N \leq 10^4$
  • Terdapat strategi yang membuat pemain pertama dapat memenangkan permainan

Subsoal 1 (0 poin)

  • $K = 1$

Subsoal 2 (0 poin)

  • $K = 2$
  • $1 \leq A_i \leq 100$
  • $1 \leq N \leq 100$

Subsoal 3 (0 poin)

  • $K = 2$

Peringatan

Selalu lakukan flush setiap kali setelah program Anda mengeluarkan keluaran.

  • Pascal:
    flush(output);
  • C/C++:
    • untuk library stdio.h (cstdio):
      fflush(stdout);
    • untuk library iostream:
      std::cout << std::flush;
