Description

Given an integer $N$. You are asked to output any $4$ non-negative integers $a$, $b$, $c$, and $d$, such that the value of $a^2 + b^2 + c^2 + d^2$ is as close to $N$ as possible. Your score will depend on the difference between $a^2 + b^2 + c^2 + d^2$ and $N$, using the formula that will be explained in the Scoring section.

Problem Type Information

This is an "output-only" problem. For each test case, you write the output to a file.

The input files for this problem can be downloaded in the Attachment section at the end of this problem. Inside the .zip file, there are 1 + 10 input files to solve: empat_contoh_1.in, empat_1.in, empat_2.in, ..., empat_10.in. The sample input (empat_contoh_1.in) are not included in the final score.

For each input file that you solve (you don't have to solve all input files), create an output file empat_contoh_T.out (for sample input) or empat_T.out, where T is the test case number. Then, compress all output files into a single .zip file. If you made any program to help solving this problem, include all of your source codes into the .zip file, then submit.

Input Format

The input is given with the following format: 

N

Output Format

A single line consists of $4$ integers $a$, $b$, $c$, and $d$.

Explanation of Sample 1

In this sample, the value of $a^2 + b^2 + c^2 + d^2$ is $4^2 + 3^2 + 2^2 + 2^2 = 33$. Using the formula given in the Scoring section, the score achieved by this sample output is $10$.

Penilaian

If $X$ is the value of $a^2 + b^2 + c^2 + d^2$, then the score you achieved is:

Condition Points
$|N-X| \geq 10$$0$
$|N-X| \lt 10$$10 - |N-X|$

The notation $|a|$ returns absolute value of $a$. For every input, it can be proven that there exists an output that achieves score $10$.
