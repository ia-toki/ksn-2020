Description

Define a recursive function $f(x,y)=x^{f(x,y-1)}$ if $y \gt 0$, and $f(x,y)=1$ if $y = 0$. Given $3$ positive integers $A$, $B$, and $M$, determine the value of $f(A,B)$ modulo $M$.

Input Format

The input is given with the following format: 

A B M

Output Format

A single line consists of the value of $f(A,B)$ modulo $M$.

Sample Input 1

2 3 5

Sample Output 1

1

Explanation of Sample 1

$f(2,3) = 2^{f(2,2)} = 2^{2^{f(2,1)}} = 2^{2^{2^{f(2,0)}}} = 2^{2^{2^1}} = 2^{2^2} = 2^4 = 16$.
