Deskripsi

Definisikan sebuah fungsi rekursif $f(x,y)=x^{f(x,y-1)}$ untuk $y \gt 0$, dan $f(x,y)=1$ untuk $y = 0$. Diberikan $3$ buah bilangan bulat positif $A$, $B$, dan $M$, tentukan nilai dari $f(A,B)$ modulo $M$.

Format Masukan

Masukan diberikan dalam format berikut: 

A B M

Format Keluaran

Sebuah baris berisi nilai dari $f(A,B)$ modulo $M$.

Penjelasan Contoh 1

$f(2,3) = 2^{f(2,2)} = 2^{2^{f(2,1)}} = 2^{2^{2^{f(2,0)}}} = 2^{2^{2^1}} = 2^{2^2} = 2^4 = 16$.

Subsoal

Untuk semua subsoal, berlaku:

  • $1 \leq A, B, M \leq 10^{9}$
  • $M = 2$
  • $M = 3$
  • $M = 4$
  • $M = 5$
  • $M = 6$
  • $B = 2$
